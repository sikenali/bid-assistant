# 文件对对碰 - 专业文档对比工具

## 项目概述

**文件对对碰**是一款专业的文档对比工具，基于Vue 3 + TypeScript开发，支持多种文件格式的内容对比和属性检查，并提供AI辅助分析功能。

## 技术栈

- **前端框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **路由**: Vue Router 4
- **UI组件**: 自定义组件 + Remix Icon
- **文档处理**: mammoth (Word), pdf-parse (PDF), xlsx (Excel)
- **报告生成**: docx库
- **桌面应用**: nwjs

## 核心功能

### 1. 文件对比
- 支持多种文件格式（Word、PDF、Excel、TXT等）
- 基于最长公共子串算法计算文本相似度
- 高亮显示雷同片段，支持自定义匹配规则
- 可按相似度排序和筛选结果
- 支持导出Word格式对比报告

### 2. 属性检查
- 对比文件元数据（名称、大小、类型、作者等）
- 直观展示匹配/不匹配/警告状态
- 支持查看历史对比记录
- 可导出属性对比报告

### 3. 系统设置
- 常规设置：调整查重参数（最小字数、相似度阈值等）
- 高级设置：配置AI模型和API密钥
- 支持忽略大小写、标点符号和空格差异

### 4. 最近记录
- 保存历史对比记录
- 支持查看、删除和清空记录
- 按对比类型分类管理

### 5. AI辅助分析
- 集成多种AI大模型
- 提供智能分析和建议
- 支持自定义AI模型配置

## 编译与运行

### 开发环境运行

1. 安装依赖
```bash
npm install
```

2. 启动开发服务器
```bash
npm run dev
```

3. 在浏览器中访问 `http://localhost:5173`

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录

### 打包桌面应用

```bash
npm run package
```

桌面应用打包使用nwjs，支持Windows、macOS、Linux平台

## 使用说明

### 文件对比流程

1. 点击"文件对比"选项卡
2. 上传或拖拽两个要对比的文件
3. 设置对比参数（可选）
4. 点击"开始对比"按钮
5. 查看对比结果，包括相似度、雷同片段等
6. 可导出对比报告或保存对比记录

### 属性检查流程

1. 点击"属性检查"选项卡
2. 上传或拖拽两个要检查的文件
3. 点击"开始检查"按钮
4. 查看文件属性对比结果
5. 可导出属性报告或保存检查记录

### AI辅助分析

1. 在系统设置中配置AI模型和API密钥
2. 在文件对比结果页面点击AI分析按钮
3. 等待AI生成分析结果
4. 查看AI提供的智能建议和分析

## 配置说明

### 常规设置

- **最小字数**: 设置匹配片段的最小字数阈值
- **相似度阈值**: 设置判定为雷同的相似度百分比
- **忽略大小写**: 对比时是否忽略大小写差异
- **忽略标点符号**: 对比时是否忽略标点符号
- **忽略空格**: 对比时是否忽略空格差异

### 高级设置

- **AI模型**: 选择要使用的AI模型（如GPT-4、Claude等）
- **API密钥**: 输入对应AI模型的API密钥
- **请求超时**: 设置AI请求的超时时间
- **最大 tokens**: 设置AI响应的最大tokens数
