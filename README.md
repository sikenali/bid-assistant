# 猩文标聚 

文档智能处理平台 — 取、制、检、版、比，五步成章，回旋有术。


## 产品架构

平台围绕"文房五宝"五大核心能力构建，每张卡片代表一项文档智能服务：

| 卡片 | 名称  | 功能   | 说明                                      |
| -- | --- | ---- | --------------------------------------- |
| 1  | 文取猩 | 智能采集 | 从各类来源提取文档内容，支持 URL 采集、文件上传、API 对接、数据库导入 |
| 2  | 文制猩 | 智能生成 | 根据模板和参数自动生成标准化文档                        |
| 3  | 文检猩 | 智能校验 | 对文档内容进行合规性、完整性校验                        |
| 4  | 文版猩 | 智能排版 | 自动调整文档格式、字体、间距等排版要素                     |
| 5  | 文比猩 | 智能比对 | 对比多份文档差异，高亮标注变更内容                       |

## 页面说明

- **首页**：品牌展示 + 五宝卡片入口
- **投标网站**：常用招投标平台快捷入口，支持一键复制网址
- **信用资质**：企业资质信息管理
